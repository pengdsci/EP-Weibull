Generating and Characterizing Probability Distributions

A Series of Research Tutorials

[Last updated: 6/3/2024]

This web page provides a series of tutorials that lay out the general framework for generating and characterizing new probability distributions with some special discussions on lifetime distributions. Most of the tutorials were drafted using RMarkdown. The following is a list of tutorials with RMarkdown source code, HTML and PDF of each individual tutorial except for the first one. There are a lot of free online resources avaialble. The Statlec site has series notes that systematically cover probability, statistics, and some machine learning. HTML Link .
